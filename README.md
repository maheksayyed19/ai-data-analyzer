# 🚀 AI Data Analysis Tool

An AI-powered web app that analyzes any CSV file and generates insights automatically.

---

## 🌟 Features

- 📂 Upload any CSV file  
- 🤖 AI-generated insights (on demand)  
- 📊 Smart visualizations  
- ⚡ Clean and simple interface  

---

## 🧠 How it works

1. Upload your CSV file  
2. Click "Generate AI Insights"  
3. View trends, patterns, and suggestions  

---

## 🛠️ Tech Stack

- Python  
- Streamlit  
- Pandas  
- Plotly  
- FastAPI  
- Groq (LLM)  

---

## ▶️ Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
